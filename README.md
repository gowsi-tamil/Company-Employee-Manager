# Employee Management
CRUD app - Employee Management (Mean)

# Prerequisite:
 -> Angular CLI: 15.0<br/>
 -> Node: 16.17.0<br/>
 -> MongoDB - latest

# Steps:
1. Create .env file in backend and set below values<br/>
 PORT=...<br/>
 MONGO_URL=...<br/>

2. Start Backend
 - cd backend
 - npm install
 - npm start
3. Start Frontend
 - cd frontend
 - npm install
 - ng serve
